EasyAdmin Casino Ratings Application
==========================

Requirements
------------

  * PHP 7.3 or higher;
  * PDO-SQLite PHP extension enabled;
  * and the [usual Symfony application requirements][2].

Installation
------------

Run this command with [Composer][4]:

```bash
$ composer install
```
